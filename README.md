# Kirana-AI 🌾🤖

**Empowering the backbone of Indian retail with Organic Intelligence.**

Kirana-AI is a next-generation quick-commerce platform designed specifically for the Indian Kirana store ecosystem. It bridges the gap between traditional local grocery stores and modern digital shopper expectations through a premium, high-tech experience.

> [!NOTE]
> This project is built using the **Liquid Glass** design system, focusing on high-end aesthetics (Glassmorphism) and smooth, data-driven interactions.

---

## ✨ Key Features

- **Liquid Glass UI**: A state-of-the-art glassmorphic design system for a premium user experience.
- **Organic Intelligence Branding**: A unique visual identity merging traditional grain motifs with digital pulse signals.
- **Dual Perspective**: Optimized workflows for both **Store Owners** (inventory/insights) and **Customers** (shopping/express delivery).
- **Location-Aware Shopping**: Real-time store discovery based on user proximity.
- **Micro-Retail Empowerment**: Tools designed to digitize local inventory without overwhelming complexity.

## 🎨 Design Philosophy

The brand identity revolves around **"Organic Intelligence"**. The custom-designed **Hybrid Pulse-Grain** logo represents:
- **The Grain Silhouette**: Traditional Indian grocery (Kirana) heritage.
- **The Pulse Handle**: Modern digital connectivity and AI efficiency.
- **Digital Pixels**: Integrated cutouts signifying data-driven intelligence.

## 🛠️ Tech Stack

- **Platform**: [Expo SDK 54](https://expo.dev/) (React Native)
- **Router**: [Expo Router](https://docs.expo.dev/router/introduction/) for file-based navigation.
- **State**: [Zustand](https://github.com/pmndrs/zustand) for lightweight, fast state management.
- **Animation**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/) for fluid micro-interactions.
- **Visuals**: SVG-based vector components via `react-native-svg` for infinite scalability.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- pnpm (recommended) or npm
- Expo Go app on your mobile device (for testing)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/[username]/kirana-ai.git
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Start the development server:
   ```bash
   pnpm dev
   ```

## 📂 Project Structure
- `/app`: Expo Router screens and layouts.
- `/components`: Reusable UI components (Glassmorphism-ready).
- `/design-system`: Core tokens, color palettes, and styling utilities.
- `/store`: Zustand state definitions.
- `/server`: Background services and API mockups.
